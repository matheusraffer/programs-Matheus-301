Desafio: Console de Áudio High-End

Programe o controle de volume protegendo o hardware:

  • Use o campo privado #volume.
  • O Setter deve limitar o volume entre 0 e 100.
  • Se o volume for 0, o Getter deve retornar "MUTE".
  • O Setter deve atualizar o visor automaticamente.
